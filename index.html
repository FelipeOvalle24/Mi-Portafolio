MIS PROYECTOS

YEDA COSMETICS

Esta es una pagina web que desarrolle para un familiar la cual trata sobre la comercialización y venta de productos de maquillaje y sobre todo la venta de su producto estrella cajas sorpresas. en este proyecto se puede pedir cualquier producto por medio de WhatsApp. ademas de esto la web cuenta con un estilo divertido y minimalista ademas de estar adaptada tambien para dispositivos moviles, esta pagina fue desarrollada utilizando HTML, CSS y JAVASCRIPT.

CALCULADORA DE PROPINAS

Este es un sitio web que desarrolle como proyecto personal, el cual trata de una calculadora de propinas, tiene un interfaz moderna y amigable para que cualquier persona pueda usarla, esta calculadora te pide el monto de la cuenta y te da opciones de porcentajes para calcular la propina, esta web fue desarrollada utiliznado HTML, CSS y JAVASCRIPT ademas esta adaptada para dispositivos moviles.

PLANETAS DEL SISTEMA SOLAR

Este es un sitio web interactivo sobre los planetas del sistema solar para comenzar el viaje debes elegir un planeta darle click he iras a ese planeta, conoceras sus principales datos y estructura interna, este sitio web fue desarrollado utilizando HTML, CSS y JAVASCRIPT ademas esta adaptado para dispositivos moviles.

CONECTA CUATRO

Este es un video juego web en el que deberas conectar 4 fichas de manera vertical, horizontal o diagonal para ganar, fue desarrollado utilizando principalmente JAVASCRIPT, tambien se utilizo HTML y CSS, ademas esta adaptado para dispositivos moviles.

HEALTHY PETS

Este es un sitio web sobre datos y cuidados de diferentes razas de perros en el podras escojer la raza de tu perro y la pagina te dira datos de tu mascota y sus cuidados, este sitio web fue desarrollado utilizando HTML, CSS y JAVASCRIPT ademas esta adaptado para dispositivos moviles.

CIUDADES DE COLOMBIA

Este es un sitio web con los datos de temperatura promedio, altura sobre el nivel del mar y total de poblacion de las ciudades mas importantes de colombia, este sitio web fue desarrollado utilizando HTML, CSS y JAVASCRIPT.

INTERACTIVE SNAKE
